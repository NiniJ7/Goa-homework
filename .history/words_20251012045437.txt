syntax - წერის წესები
() - ფრჩხილები, parenthesis
'' / "" - ბრჭყალები, quotation marks
variable - ცვლადი
variable name - ცვლადის სახელი
variable value - ცვლადის მნიშვნელობა
paragraph - ტექსტი
header - სათაური
memory - მეხსიერება
output - მნიშვნელობის გამოტანა
numerical value - რიცხვითი მნიშვნელობა
declare - დეკლარირება/შექმნა
execeute - გაშვება
debugging - შეცდომების გამოსწორება
store - შენახვა
frontend - საიტის ვიზუალური მხარე, რასთანაც შეხება აქვს მომხმარებელს
Backend - საიტის ფუნქციონალური მხარე, რომელსაც ჩვენ ვერ ვხედავთ
HTML - ტექნოლოგია, რომელიც გამოიყენება საიტის სტრუქტურის შესაქმნელად
CSS - ტექნოლოგია, რომელიც გამოიყენება საიტის დიზაინის შესაქმნელად
Javascript - ენა, რომლითაც ჩვენი საიტები ხდება ფუნქციონაალური,ინტერაქციული
interactivity - მოძრავი
< > - angle brackets, კუთხური ფრჩხილები
convert - გადაყვანა
anchor - ღუზა
